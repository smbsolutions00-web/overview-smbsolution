# SMB Solutions Opportunity Meeting

Landing page for the SMB Business Opportunity meeting.

Live URL: https://overview.smbsolution.ai (Render Web Service, Starter plan)

## Run locally
```
npm start
```
Then open http://localhost:10000

## Files
- `index.html` — Landing page with Zoom registration CTAs
- `images/` — Hero and content images
- `server.js` — Minimal zero-dependency Node static server (for Render Web Service)
- `package.json` — Node project metadata

## Deploy
Hosted as a Render Web Service. Start command: `npm start`. Build command: none (no dependencies).
